# Categoria: Admin y CMS

## Implementado

- Dashboard con metricas operativas.
- Editor de articulos con:
  - SEO avanzado.
  - Multimedia.
  - Autosave.
  - Historial de versiones.
  - Vista previa.
- Moderacion de comentarios:
  - aprobar/rechazar/eliminar,
  - editar comentario,
  - marcar spam,
  - bloquear usuario,
  - responder desde admin.
- Gestion de usuarios:
  - roles (Admin, Editor, Moderador, Usuario),
  - suspender/expulsar/reactivar,
  - editar perfil.
- Gestion de comunidad:
  - cerrar/fijar/eliminar temas,
  - moderar respuestas.

## Notas

- Se uso persistencia combinada: Supabase + respaldo local para campos/modulos no modelados en DB.
